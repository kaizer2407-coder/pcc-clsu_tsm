# PCC CLSU Travel System Management (TSM)

A web-based **Travel System Management System** built using Laravel. This system helps manage travel requests, driver assignments, approvals, and monitoring for organizational use.

---

## 🚀 Features

- Travel Request Submission  
- Admin Approval System  
- Driver Management  
- Travel Scheduling  
- Dashboard Overview  
- Active Travel Monitoring (based on date)  
- Status Tracking (Pending / Approved / Completed)  
- Search and Filter (Passenger, Date, Status)

---

## 🛠️ Technologies Used

- PHP (Laravel Framework)  
- MySQL / PostgreSQL  
- HTML, CSS, Bootstrap  
- JavaScript
